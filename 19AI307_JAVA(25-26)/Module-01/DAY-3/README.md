# Ex.No:1(C) LOOPING STATEMENT

## AIM:

Repeat an action multiple times using a loop structure.

## ALGORITHM :

1. Start the program.
2. Initialize a counter variable.
3. Use a loop to run while the counter is within a limit.
4. Print the counter value each time the loop runs.
5. End the program once the loop finishes.

## PROGRAM:

#### Register Number: 212223240183

### to implement a Looping Statement using Java

## SOURCE CODE:

```java
public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
        }
    }
}
```

## OUTPUT:

```
1
2
3
4
5
```

## RESULT:

Program executed successfully.
