## AIM:

Store multiple values in a single variable using an array and display them.

## ALGORITHM :

1. Start the program.
2. Declare and initialize an array with a few elements.
3. Use a loop to access each element.
4. Print every element during iteration.
5. End the program after printing all values.

## PROGRAM:

### to implement a Array concept using Java

## SOURCE CODE:

```java
public class Main {
    public static void main(String[] args) {
        int[] nums = {10, 20, 30, 40, 50};
        for (int n : nums) {
            System.out.println(n);
        }
    }
}
```

## OUTPUT:

```

20
30
40
50
```
